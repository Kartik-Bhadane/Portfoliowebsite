Face Mask Detection Using Python

This project develops a Python-based system for real-time face mask detection, utilizing computer vision and deep learning techniques. A Convolutional Neural Network (CNN) is trained on labeled images to accurately classify whether individuals in video streams are wearing masks or not. The implementation employs libraries such as OpenCV and TensorFlow/Keras, enabling the model to process video input from webcams or surveillance cameras. This solution is designed to help enforce mask-wearing policies in public spaces, contributing to public health efforts during the COVID-19 pandemic.

YouTube Clone

This project involves the development of a YouTube clone, a web application that mimics the core functionalities of YouTube. Built using modern web technologies such as HTML, CSS, JavaScript, and a backend framework like Django or Node.js, the platform allows users to upload, view, and share videos. Key features include user authentication, video streaming, search functionality, comment sections, and video recommendations. The project demonstrates the implementation of scalable, media-rich web applications and offers insights into managing video content, user interactions, and server-side processing..
